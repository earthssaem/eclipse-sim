# 🌑 일식 · 월식 탐구 시뮬레이터

태양 · 지구 · 달의 위치 관계에 따른 식(蝕) 현상을 직접 조작하며 탐구할 수 있는 교육용 웹 시뮬레이터입니다. 
중학교 과학, 고등학교 지구과학 교과 수업에 활용하기 위해 제작하였습니다.

## ✨ 주요 기능

### 일식 모드
- 달의 **본그림자(umbra)**, **반그림자(penumbra)**, **금환 영역(antumbra)** 의 실제 기하학적 콘을 외접선·내접선으로 정확히 표현
- 본그림자 꼭짓점은 외부 닮음중심 `D·r/(R−r)` 위치에 형성
- 지구 표면의 관측 지점에 빨간 막대사탕 마커 📍 를 꽂아 드래그 가능
- 관측 위치에 따라 **개기일식 / 부분일식 / 금환일식 / 일식 없음** 을 자동 판정
- 지구 표면을 따라 관측자의 위치별 식 종류가 색으로 표시 (낮 반구만)

### 월식 모드
- 지구의 본그림자·반그림자에 들어간 달의 상태를 자동 판정
- **개기월식 / 부분월식 / 반영식 / 월식 없음** 표시
- 개기월식 시 달이 블러드문(붉은빛) 으로 표현

### 관측자 시점 인셋
- 우측 상단에 96px 인셋으로 **그 위치에서 실제로 보이는 모습** 을 실시간 렌더링
- 일식
  - 개기일식: 검은 달 원반 + 은빛 코로나(아지랑이) + 분홍 채층(애니메이션)
  - 금환일식: 실제 시지름 비율로 축소된 달 → 두꺼운 태양 고리
  - 부분일식: 시지름 비교 + 관측자 위치에 따른 부분 가림
- 월식
  - 부분/반영식: 차분한 회색 음영으로 가려진 달
  - 개기월식: 블러드문

### 조작 인터페이스
- ▶ 공전 시작 / ↺ 초기 위치
- 달까지 거리 슬라이더 (0.45×~1.7×)
- 공전 속도 슬라이더 (매우 느림 ~ 매우 빠름)
- 달과 관측 위치를 마우스/터치로 직접 드래그

## 🧭 사용법

1. 상단 탭에서 **🌑 일식** 또는 **🌕 월식** 모드 선택
2. 마우스로 **달** 또는 **관측 위치(빨간 막대사탕)** 을 드래그
3. 슬라이더로 **달까지 거리** · **공전 속도** 조절
4. **▶ 공전 시작** 으로 달이 지구 주위를 돌게 두고 식 현상이 일어나는 순간 관찰
5. 하단 상태 카드에서 현재 식의 종류와 설명, **관측자 시점** 인셋에서 실제 보이는 모습 확인

## 🚀 실행

별도의 빌드 과정 없이 정적 HTML 파일 하나로 동작합니다.

```bash
# 직접 열기
open index.html
# 또는 간단한 정적 서버
python3 -m http.server 8000
```

브라우저에서 `index.html` 을 열기만 하면 됩니다. 외부 의존성은 Google Fonts(Noto Sans KR) 뿐이고 오프라인에서도 폰트만 fallback되며 정상 동작합니다.

## 🛠 기술 스택

- 순수 **HTML5 Canvas + Vanilla JavaScript**
- 빌드 도구·프레임워크·외부 라이브러리 없음
- 단일 파일 (`index.html`) 로 모든 로직·스타일·자산 포함

## 🎓 교육적 정확성

- 본그림자 경계 = 외접선 (외부 닮음중심에서 tip 형성)
- 반그림자 경계 = 내접선 (관측자 입장에서 발산)
- 반그림자 폭 = `r + L·(R+r)/D`
- 관측자 시점의 달 · 태양 시지름 비율은 실제 거리 기반 `R/D` 로 계산
- 지구의 낮 반구(태양을 향한 면) 에만 식 표시 → 밤 반구로 그림자가 새지 않음

## 📐 시뮬레이션 좌표계

- 캔버스 900 × 380 px
- 태양 (110, 190), 반지름 52
- 지구 (640, 190), 반지름 28
- 달 공전 궤도 타원: `BASE_OX=150, BASE_OY=150`
- 달 반지름 13

> 천체 크기·거리는 시각화 목적의 비율이며 실제 천문학적 비율은 아닙니다.
